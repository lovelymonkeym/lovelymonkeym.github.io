    示例：日期格式化 ``` months = [ 'January', 'February', 'March', 'April', 'May', 'June', 'July', 'August', 'September', 'October', 'November', 'December' ]

    endings = ['st','nd','rd'] + 17['th']\ + ['st','nd','rd'] + 7['th']\ +['st']

    year = input('Year:') month = input('Month(1~12):') day = input('Day(1~31):')

    month_number = int(month) day_number = int(day)

    month_name = months[month_number - 1] ordinal = day + endings[day_number - 1]

    print(month_name + ' ' + ordinal + ' ' + year)

    执行结果

    Year:1990 Month(1~12):6 Day(1~31):26 June 26th 1990

    `` - ####分片#### + 使用冒号 + 提供两个索引边界，第一个包含在分片内，第二个不包含在分片内 + 如果右边的索引出现在左边的索引之前，将输出空[]+ 如果分片所得部分包含列表首尾元素，只需将相应索引置空[index1:] or [:index2]+ 步长：[index1:index2:step]`,默认为1。为负数时从右向左提取元素,此时，index2要小于index1

    示例1： ```

    对http://www.something.com形式的url做截取处理

    url = raw_input('Input a url:') print('Second Level Domain:' + url[11:-4]) print('Top Level Domain:' + url[-3:])

    执行结果

    Input a url:http://www.baidu.com Second Level Domain:baidu Top Level Domain:com
