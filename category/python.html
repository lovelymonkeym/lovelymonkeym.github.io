基本的列表操作

  • 赋值：使用下标
  • 删除：del list[index]
  • 分片赋值：list_1[index_1:index_2] = list_2[member_1,member_2,...]
  • 通过分片赋值语句插入、删除元素: #在index的位置插入list_2 list_1 = [member_1,member_2,...] list_2 = [m1,m2,...] list_1[index:index] = list_2 #删除index_1到index_2之间的元素 list_1[index_1:index_2] = []
